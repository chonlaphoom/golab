# RPC Experiments

Experiments with Remote Procedure Call (RPC) implementations in Go.

## Structure

- `gRPC/` - gRPC implementation experiments
- `twirp/` - Twirp RPC framework experiments
- `cmd/` - Command-line tools and examples

## Overview

This module contains experiments with different RPC frameworks and patterns:

- **gRPC** - Google's high-performance RPC framework
- **Twirp** - Simple RPC framework using Protocol Buffers

## Requirements

- Go 1.25.0+
- Protocol Buffers compiler (protoc) for code generation

## Usage

Each subdirectory contains specific RPC implementation examples.
